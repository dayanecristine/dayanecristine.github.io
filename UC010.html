Diagrama de caso de uso UC010

Um caso de uso é um documento narrativo que descreve a sequência de eventos de um ator que usa um sistema para completar um processo.

Listar vendedores - UC010

O ator deve ser capaz de listar os vendedores no sistema.

Ator:
  • Consumidor
Pre-Condições:
  • O ator deve estar logado no sistema.
  • O ator deve possuir permissão de listar vendedores.
  • O ator deve estar na página de listar vendedores.
Pos-Condições:
  • Os vendedores devem estar listados no sistema.
