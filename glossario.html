Glossário

Palavras-chave para ajudar o entendimento do projeto.

Requisitos - Requisitos são exigências que representam uma ou mais funcionalidades de um projeto.
Requisitos funcionais - Requisição que um software deverá materializar.
Requisitos não funcionais - restrições que o sistema deverá atender, mas não realizadas através de funcionalidades.
Atores - Entidade que interage com o sistema.
Marketplace - Modelo virtual de compras, vendas e trocas de serviço.
